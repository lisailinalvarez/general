Procesamiento del Lenguaje Natural¶

Dataset: The Multilingual Amazon Reviews Corpus¶

Crditos por utilizacin de datos:
Justifying recommendations using distantly-labeled reviews and fined-grained aspects
Jianmo Ni, Jiacheng Li, Julian McAuley
Empirical Methods in Natural Language Processing (EMNLP), 2019.

Exploración de datos y Procesamiento del Lenguaje Natural¶

En este informe se trabaja sobre las crticas de productos de amazon y se desarrolla un modelo que asigna la cantidad de estrellas correspondiente a la review. Se divide en tres partes: primero, se hace una pequea exploracin de los datos. Luego se implementa ya el modelo adecuado elegido y finalmente se presentan recomendaciones para mejorar el trabajo realizado.

En esta parte nos dedicaremos a hacer una pequea exploracin de datos. Notaremos, cules son las palabras ms utilizadas para las crticas positivas, negativas y neutras?

# Recorremos todos los titulos y le vamos aplicando la Normalizacion y luega el Stemming a cada uno
review_list_train=[]
for review_train in dataset_train.review_body:
    # Vamos a reemplzar los caracteres que no sean leras por espacios
    review_train=re.sub("[^a-zA-Z]"," ",str(review_train))
    # Pasamos todo a minsculas
    review_train=review_train.lower()
    # Tokenizamos para separar las palabras del titular
    review_train=nltk.word_tokenize(review_train)
    # Eliminamos las palabras de menos de 3 letras
    review_train = [palabra for palabra in review_train if len(palabra)>3]
    # Sacamos las Stopwords
    review_train = [palabra for palabra in review_train if not palabra in stopwords]
    
    ## Hasta ac Normalizamos, ahora a stemmizar
    
    # Aplicamos la funcion para buscar la raiz de las palabras
    review_train=[stemmer.stem(palabra) for palabra in review_train]
    # Por ultimo volvemos a unir el titular
    review_train=" ".join(review_train)
    
    # Vamos armando una lista con todos los titulares
    review_list_train.append(review_train)
    #dataset["titular_normalizado"] = titular_list
#___________________________________________________________________________________________
review_list_test=[]
for review_test in dataset_test.review_body:

from sklearn.metrics import confusion_matrix

def confusion(y_test,y_pred):
    names=["1","2","3","4","5"]
    cm=confusion_matrix(y_test,y_pred)
    f,ax=plt.subplots(figsize=(5,5))
    sns.heatmap(cm,annot=True,linewidth=.5,linecolor="r",fmt=".0f",ax=ax)
    plt.xlabel("y_pred")
    plt.ylabel("y_true")
    ax.set_xticklabels(names)
    ax.set_yticklabels(names)
    plt.show()

    return
